Lua VM 混淆器：強度選項版

新增功能：
- 基礎 VM 混淆
- 高強度混淆
- 陷阱混淆

資料夾內容：
- index.html
- style.css
- script.js
- README.txt

使用：
1. 打開 index.html。
2. 選擇 .lua 或 .txt。
3. 勾選需要的混淆選項。
4. 按開始 VM 混淆。
5. 按下載混淆結果。

選項說明：
- 基礎：最穩，輸出較小。
- 高強度：加入二次數字變形，輸出比較難看懂。
- 陷阱：加入假指令和假分支，檔案會更大，建議測試後再使用。

注意：
- 混淆後的 Lua 仍需要目標環境支援 loadstring 或 load。
- 如果目標環境禁止 loadstring/load，就不能執行。
- 高強度和陷阱會讓輸出檔案變更大。
